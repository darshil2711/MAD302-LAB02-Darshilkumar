## Project Description

An Android application that displays a list of products using RecyclerView.
Users can tap any product to navigate to a DetailActivity that shows full
product information. Demonstrates click handling, Intent-based navigation,
and passing data between activities using Kotlin.

## Features
- Displays 6+ products with name, price, and description
- RecyclerView with custom adapter
- Lambda-based click handling
- Navigation to detail screen via Intent
- Null-safe data handling on detail screen

## Tech Stack
- Language: Kotlin
- UI: RecyclerView, CardView
- Navigation: Android Intent
